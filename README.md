# Calculator – React

A simple calculator built using React to practice **state management, event handling, and component communication**.

This project focuses on understanding how user interactions update UI through React state.

---

## Features
- Button-based calculator input
- Supports basic arithmetic operations (`+`, `-`, `*`, `/`)
- Clear (`C`) button to reset the display
- Result evaluation using `=` button
- Read-only display controlled by React state
